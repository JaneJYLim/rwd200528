[형상관리 프로그램 또는 기법]
    1. CVS
    2. SVN
    3. GIT(깃) 
       => 가장 최근에 만들어진 버전 관리 프로그램
        리눅스 운영체제(Linux) 제작자가 만듦.
        명령어가 리눅스 명령어를 사용한다.
        퍼블리셔, 프론트엔드 개발자, 백엔드 개발자는 리눅스 명령어를 사용하지 않고, 편집기 또는 IDE를 사용하여 GIT을 사용함.
        최종 사용 목적: GIT이라는 기술을 사용하여 인터넷 공간의 서버에 만들어둔 프로그램을 날짜와 시간, 버전, 기타 서술내용을 포함하여 저장하는 것.
        GIT 기술을 사용하여 무료로 제공되는 버전을 구분하여 저장할 수 있는 인터넷 저장공간에 저장하는 것을 최종목적으로 한다.
        필요한 과거자료는 언제나 다운로드 받을 수 있다.
        이 때 사용하는 인터넷 저장공간 : Github(깃허브)

[Git과 Github를 사용한 프로그램 형상관리]
1. 관련 프로그램 설치
    1) Git 프로그램 다운로드 및 설치
       다운로드 인터넷 주소 => https://git-scm.com
    2) 다운로드 파일 저장위치 => D:\FrontEndTool\download\scm\Git
    3) 설치위치 => D:\FrontEndTool\installed\Git

2. 인터넷 저장 사이트 회원가입 및 로그인 후
   1) 저장소 회원가입 후 로그인
       => Github 사이트 회원가입 후 저장소 만들어서 제작한 프로그램 버전별로 또는 날짜별로 저장하여 관리하기.
        깃허브 주소 => https://www.github.com
        
    2) 저장소(=리파지터리) 생성하기
        => Create repositiory... 클릭
        => 저장소 주소
            http://github.com/본인계정/저장소 이름.git
            
            보기. 내꺼 github주소
            http://github.com/JaneJYLim/rwd200528.git (내꺼)
            
3. 브라켓에서 Git 사용하기
    1) 플러그인 확장에서 Git 설치
        플러그인 확장 관리자 클릭
        => "설치가능" 탭에서 git 입력
    2) 설치항목 : Brackets Git
    3) 플러그인 설치 후 설정 관련사항은 모두 기본값으로 설정 or 모두 cancel 버튼 선택하면 기본값으로 설정됨.
    4) 우측 상단 아이콘 목록 영역에 Git 아이콘 생성 확인
        => 클릭하면 하단에 git 메뉴화면 활성화됨.
